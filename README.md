# 🏥 Sistema de Consultas - TypeScript
 
Projeto desenvolvido com o objetivo de simular um sistema básico de agendamento de consultas médicas, utilizando TypeScript e boas práticas de organização de código.
 
A aplicação contempla a modelagem de dados como especialidades, médicos, pacientes e status de consultas, com foco em tipagem e estruturação.
 
---
 
## 📌 Objetivo
 
Aplicar na prática conceitos fundamentais de TypeScript:
- Interfaces
- Types (Union Types)
- Organização de código
- Estruturação de dados
 
---
 
## ⚙️ Tecnologias utilizadas
 
- TypeScript
- Node.js
- ts-node
 
---
 
## 📁 Estrutura do projeto
 
src/
├── types/
│    ├── especialidade.ts
│    ├── paciente.ts
│    ├── statusConsulta.ts
│
├── interfaces/
│    ├── medico.ts
│    ├── consulta.ts
│    ├── usuarioAdmin.ts
│
├── index.ts
 
---
 
## 🚀 Como executar o projeto
 
1. Clone o repositório:
git clone https://github.com/ythurzinx/sistemas-consultas-typescript
 
2. Acesse a pasta:
cd sistemas-consultas-typescript
 
3. Instale as dependências:
npm install
 
4. Execute o projeto:
npx ts-node src/index.ts
